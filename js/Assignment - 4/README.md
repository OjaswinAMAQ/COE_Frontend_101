# Employee Skill Tracker Dashboard

A dynamic HR dashboard built using pure HTML, CSS, and JavaScript.

## Features
- Employee registration with validation
- Skill tracking
- Certification status
- Dynamic filtering
- Live stats
- LocalStorage persistence
- Responsive UI

## JavaScript Concepts Used
- Variables & data types
- Arrays, objects, reduce, map, filter
- DOM manipulation
- Event handling
- Form validation
- Local storage

## How To Run
1. Open index.html in any browser
2. Add employees via the form
3. Filter, delete, and view stats in real-time
