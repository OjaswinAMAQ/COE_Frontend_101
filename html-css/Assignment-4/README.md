# SmartCity Services Dashboard

## Overview
A responsive, accessible, multi-page SmartCity dashboard built using **HTML5** and **CSS3**. This dashboard allows citizens to explore city services, view alerts, watch mayoral messages, access transport information, and report issues efficiently.

## Pages
- **Home**: Hero section, alerts ticker, quick links, mayor’s video, and informational cards.  
- **Transport**: Timetables, transport cards, and placeholders for maps and weather.  
- **Report Issue**: Accessible and validated form with checkboxes, hints, and status messages.  
- **Services**: Overview of city departments, audio/video media cards, service feature highlights, and geolocation placeholders.

## Design Approach
- **Mobile-first** design ensures usability across devices.  
- **CSS Grid & Flexbox** used for layout and responsive structures.  
- **Reusable components** like cards, media sections, and buttons reduce redundancy.  
- **Consistent theming** using CSS variables for colors, radius, and typography.  
- **Page-specific styling** maintains visual distinction while keeping base styles uniform.

## Accessibility Highlights
- Semantic HTML structure for clarity and screen-reader compatibility.  
- ARIA roles applied where necessary.  
- Fully navigable via keyboard.  
- Media captions included for videos.  
- Form elements provide visual feedback for validation.

## Technologies
- **HTML5** for semantic structure  
- **CSS3** for styling, layout, and responsive design  
- No JavaScript is used; all functionality relies on HTML and CSS.
