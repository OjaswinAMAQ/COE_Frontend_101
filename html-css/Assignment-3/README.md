## Overview

NovaFlow is designed to help enterprise teams automate workflows, track real-time analytics, and scale operations efficiently. The landing page showcases:

- Hero section with call-to-action  
- Features section highlighting key platform capabilities  
- Pricing section with multiple tiers  
- Testimonials from global customers  
- Contact / sign-up form (CTA)  
- Sticky header and smooth scroll navigation  
- Fully responsive design for mobile, tablet, and desktop  

---

## Features

- **Workflow Automation:** Visual workflow builder with no-code configuration.  
- **Real-Time Analytics:** Track performance, bottlenecks, and ROI instantly.  
- **Enterprise Security:** Role-based access and encryption.  
- **Seamless Integration:** Connect with existing enterprise tools.  
- **Responsive Design:** Optimized layout for all screen sizes.  
- **Accessible:** Semantic HTML, focus-visible indicators, ARIA labels.
